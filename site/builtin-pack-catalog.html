How To Read This

Each card is one built-in pack. The Commands list is the unique top-level when.command set across the pack's structural patterns. The Subcommands list is the unique when.subcommand set when the pack uses subcommand matching.

Important caveat Some patterns depend more on flags and arguments than subcommands, so this page is a compact browsing aid, not a full rule dump. The underlying policy still lives in the JSON pack files.
